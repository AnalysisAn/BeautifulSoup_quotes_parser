# Парсинг страницы и анализ данных
Учебный проект по небольшому анализы данных, полученных при парсинге HTML-страницы с помощью BeautifulSoup.
## Цель проекта
Получить данные с веб-страницы с помощью `requests` и `BeautifulSoup`, преобразовать их в табличный формат, сохранить в CSV-файл и выполнить базовый анализ цитат, авторов и тегов.
## Данные
В проекте используется страница сайта, на которой представлен сборник цитат известных авторов. Ссылка на сайт: https://quotes.toscrape.com
## Что сделано в проекте
- проведен веб-скрейпинг страницы;
- выполнен парсинг HTML-страницы с помощью BeautifulSoup;
- собраны данные о цитатах, авторах и тегах в таблицу
- прозведен базовый анализ данных: количество цитат по авторам, частота повторений тегов, количество уникальных авторов и тегов
- загружена таблица с данными о странице
## Используемые инструменты
- Python
- Pandas
- BeautifulSoup
- Jupyter Notebook
